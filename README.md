## 📋 Available Instruction Pages

## How to Use

1. Click on any instruction above to view its full details.
2. Copy the **Quick Start Prompt** at the bottom of that page.
3. Replace the placeholder with your own idea/problem/goal.
4. Paste it into your favorite agent framework and run.

All instructions are **open source** and free to use, modify, or integrate into your projects.

## Contributing

Thank you for considering a contribution! This project thrives on community improvements.

### Ways to Contribute
- Fixing typos, grammar, or formatting
- Improving clarity or adding better examples
- Suggesting new use cases or enhancements
- Adding entirely new instruction pages
- Improving tags, structure, or documentation

### Contribution Guidelines

1. **Fork** this repository and create a new branch for your changes.
2. Follow the existing style:
   - Clear, professional, and engaging language
   - Consistent markdown formatting (headings, bullets, code blocks)
   - Include a **Best For** section, **How to Use** protocol, **Quick Start Prompt**, and **Tags**
3. Test your changes by running the instruction in an agent framework.
4. Commit your changes with a clear message.
5. Open a **Pull Request** with a description of what you changed and why.

**We welcome both small fixes and big new ideas!**

If you have a new instruction page idea, feel free to open an issue first to discuss it.
